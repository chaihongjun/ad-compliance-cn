# 审查辅助参考

## Step 0：边界模糊引导

以下场景中审查工具可能无法独立判断是否构成广告，应主动向用户提问：

| 卡住的位置 | 典型困惑 | 应问用户的问题 |
|-----------|---------|--------------|
| 营销性（0.1） | 内容像科普但隐约在推销 | "这篇内容和发布方之间是否存在商业委托/付费合作关系？发布方是否从该商品/服务的销售中直接获益？" |
| 营销性（0.1） | 用户评价/跟帖，不确定是否有推销目的 | "该评价/跟帖的发布者是否与商家存在委托关系、是否有付费推广协议？发布后是否有销售分成机制？" |
| 媒介性（0.2） | 微信群里发的内容算媒介吗？ | "该群是公开可加入，还是仅限特定成员？是通过群发工具同步推送的还是手动发送的？" |
| 受众不特定性（0.3） | 投放给"兴趣标签人群"算特定吗？ | "是否使用了算法推荐/兴趣标签/人群画像等精准投放工具？受众在触达前是否已列为可辨识的具体个人名单？" |
| 非强制性（0.4） | 商品包装印了推荐语 | "这段文字是法规强制标注的内容，还是品牌方主动添加的营销描述？有对应的法规要求条款吗？" |
| 整体不确定 | 内容混合多种性质 | "您希望按什么场景审查？（a）从严：按最严格标准逐项审查；（b）从宽：仅审查明显违规项，标注不确定点由您自行判断）" |

## 违规等级与罚款速查

### 违规等级

| 等级 | 定义 | 对应条款 |
|------|------|---------|
| **BLOCK** | 违反 §9/§10/§15/§20/§22/§37/§40(1) 等 | §57 — 罚款 20万-100万 |
| **HIGH** | 医疗/药品/特殊类别违规、虚假广告、互联网违法 | §55/§58/互联网办法 §23-§28 |
| **WARN** | 引证/专利标注/可识别性/格式/混淆问题 | §59/反不正当竞争法 §23 |
| **UNSURE** | 信息不足以判断，需补充 | 暂无处罚，但需澄清后定级 |
| **INFO** | 轻微不合规 | 无直接罚款 |

### 核心罚款速查

| 违规类型 | 法律依据 | 罚款幅度 |
|----------|---------|---------|
| 虚假广告（一般） | 广告法 §55 | 广告费 3-5 倍；无法计算罚 20万-100万 |
| 虚假广告（累犯/严重） | 广告法 §55 | 广告费 5-10 倍；罚 100万-200万，可吊销执照 |
| §9/§10/特殊禁止类 | 广告法 §57 | 罚 20万-100万，可吊销执照 |
| 医疗/药品/特殊广告违规 | 广告法 §58 | 广告费 1-3 倍；严重 3-5 倍 |
| 引证/专利/贬低/§8违规 | 广告法 §59 | 罚 10万以下 |
| 广告不可识别 | 广告法 §59(3) | 广告发布者罚 10万以下 |
| 混淆行为 | 反不正当竞争法 §23 | 违法经营额 5 倍以下或 25万以下 |
| 虚假宣传（非广告） | 反不正当竞争法 §25 | 罚 100万以下；严重 100万-200万 |
| 有奖销售违规 | 反不正当竞争法 §27 | 罚 5万-50万 |
| 商业诋毁 | 反不正当竞争法 §28 | 罚 10万-100万；严重 100万-500万 |
| 网络不正当竞争 | 反不正当竞争法 §29 | 罚 10万-100万；严重 100万-500万 |
| 互联网禁售/禁发广告 | 互联网办法 §23→§57 | 广告费 1-3 倍；无法计算罚 20万-100万 |
| 互联网未经审查/篡改 | 互联网办法 §24→§58 | 广告费 1-3 倍；严重 3-5 倍 |
| 互联网变相发布 | 互联网办法 §25→§59(3) | 广告发布者罚 10万以下 |
| 互联网未标"广告"/关不掉/欺骗点击 | 互联网办法 §26-§27 | 处 5000-3万 |
| 互联网平台未尽义务 | 互联网办法 §29→§63 | 处 1万-5万；情节严重责令停业 |
| 直播带货违规 | 互联网办法 §28→§55/§58/§59 | 依据广告法对应条款 |
| 互联网广告一键关闭违规 | 广告法 §62(2) | 广告主罚 5000-3万 |
