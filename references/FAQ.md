# 常见疑问解答（FAQ）

### Q1：「四性」到底是什么？能用一句话说清楚吗？

**商业广告 = 为了卖东西（营销性）+ 通过媒体发布（媒介性）+ 给别人看的（受众不特定）+ 自己愿意发的（非强制）。** 四性缺一不可。

### Q2：在微信朋友圈发广告，算互联网广告吗？

**算。** 朋友圈属于互联网媒介，发布的推广内容需要满足互联网广告管理办法的各项要求。即使是个人朋友圈，如果接受商家委托发布推广内容，也构成广告发布行为。

### Q3：用了「第一」「最好」「100%」但能提供证明材料，可以吗？

**广告法 §9(3) 明文禁止「国家级」「最高级」「最佳」，没有"有证据就可以用"的例外。** 但对于「第一」「100%」等不在明令禁止清单中的用语，如果有确凿的第三方证据支持，且不产生误导，实践中风险较低。但建议谨慎使用，可以用「行业领先」「经过XX认证」等更安全的表述替代。

### Q4：我的微信公众号发了一篇健康科普文章，末尾附了医院的简介和联系方式，算医疗广告吗？

**这取决于文章的主体内容和呈现方式。** 根据《医疗广告认定指南》：
- 如果文章是客观的医学科普，末尾只标注了医院名称等基本信息 → 科普本身不认定为医疗广告（§7），附基本信息不触发 §6
- **但如果**文章末尾同时出现了推荐性语言、诊疗技术优势、预约链接或购买入口 → 可能触发 §8 变相医疗广告认定

**关键判断标准**：科普内容是客观知识介绍，还是在间接推介某家医院的服务？

### Q5：KOL 在小红书/抖音上发产品测评附购物链接，需要标「广告」吗？

**需要。** 根据互联网广告管理办法 §9(3)，以体验分享、消费测评等形式发布并附购物链接的，应当显著标明「广告」。不标注的，广告发布者可能被罚 10万元以下（广告法 §59），或 5000-3万元（互联网办法 §26）。

### Q6：直播带货过程中说「这个产品特别好用」，算广告吗？谁承担责任？

**算商业广告。** 根据互联网办法 §19：
- 商品销售者（商家）通过直播推销 → 承担**广告主责任**
- 直播间运营者提供广告设计/制作/代理 → 承担**广告经营者/发布者责任**
- 主播以自己名义推荐/证明 → 承担**广告代言人责任**

### Q7：我在商品包装上印了「品质最优」「销量领先」，这算广告吗？

**首先判断四性中的非强制性**：包装上的信息如果是法律强制标注（如生产日期、成分表），不算广告。但如果品牌方主动添加营销性描述，则满足四性，构成广告。「品质最优」属于绝对化用语风险，「销量领先」如果有数据支撑且标明了出处，风险较低。

### Q8：审查报告中同一项内容同时触犯了多个条款，以哪个为准？

**以处罚最重的条款为准**，但在报告中应列出所有被触犯的条款。例如，教育培训广告中「保过」承诺同时违反 §24(1)（教育培训专项）和 §28（虚假广告），在 BLOCK 项中应同时标注两个条款，罚款对照 §58（特殊类别违规）或 §55（虚假广告）中的较高者。

### Q9：我的广告在多个平台同时投放，管辖归属怎么确定？

根据执法指南（二）：
- **传统媒体广告** → 广告发布者所在地市监局管辖
- **互联网广告** → 广告主所在地或广告经营者所在地市监局也可管辖
- **商品包装物广告** → 违法行为发生地管辖
- 同一违法广告的多个关联主体 → 可并案处理，由最先立案的部门管辖

### Q10：如果审查结果全部 PASS，是不是就绝对安全了？

**PASS 代表依据当前提供的信息和适用法规，本工具未检测到明显违规项。** 但不代表绝对无风险，原因包括：（1）法规解释存在实践差异，不同地区的执法标准可能不同；（2）用户提供的信息可能不完整；（3）广告发布后的实际效果（如是否产生误导）可能影响认定。建议在 PASS 基础上，额外征求专业法律顾问意见。
