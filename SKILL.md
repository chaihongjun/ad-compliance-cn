# ad-compliance-cn — 中文广告合规审查

依据 8 部法规/指南对商业广告进行系统性审查。法规原文见 [references/](references/) 目录。

## 审查流程

```
广告内容（文本/图片）输入
  ├─ Step 0:  前置广告判断（执法指南一四性判断）
  ├─ Step 1:  分类识别
  ├─ Step 2:  图片与视觉内容专项审查
  ├─ Step 3:  通用禁用情形审查（广告法 §9-§10）
  ├─ Step 4:  引证与专利审查（广告法 §11-§12）
  ├─ Step 5:  虚假广告 + 虚假宣传（广告法 §28 + 反不正当竞争法 §9）
  ├─ Step 6:  反不正当竞争法专项（混淆/有奖销售/诋毁/网络不正当竞争）
  ├─ Step 7:  特殊类别专项审查（广告法 §15-§27）
  ├─ Step 8:  互联网广告专项审查
  ├─ Step 9:  医疗广告认定分析
  ├─ Step 10: 医疗广告管理办法专项
  ├─ Step 11: 违规等级与罚款区间评估
  └─ Step 12: 输出结构化报告
```

**法规原文**（审查中按需加载）：
- 广告法 / 互联网广告管理办法 / 医疗广告管理办法 / 认定指南 / 认定指南解读 / 反不正当竞争法 / 执法指南（一）（二）→ [references/](references/)

> **⚠️ 使用提示**：本工具供初步合规参考，不构成法律意见。输入超 2000 字或涉及多渠道多场景建议分段提交，或说明"从严审查"。工具无法独立判断时会提示补充信息。

## 快速开始

直接粘贴广告文案即可。如含图片请附画面描述（如"海报背景医生患者握手照片，大字'种植牙首选'，底部有医院LOGO和地址"）。审查结果分 BLOCK（必须改）/ WARN（建议改）/ INFO（参考）/ UNSURE（需补充）四级。初次使用建议先看 [案例演示](references/CASES.md)。

## Step 0 — 前置广告判断（执法指南一）

> **适用边界**：本步骤判断待审查内容是否构成「商业广告」，从而确定是否落入本技能审查范围。
> 详见 [执法指南（一）](references/广告法适用问题执法指南（一）.md)。

**四性判断**：商业广告须同时具备以下四性，缺一不可。

### 0.1 营销性 — 目的为推销商品/服务
- [ ] 是否直接或间接推销商品/服务？
- [ ] 以下情形**不认定**为营销性（不适用广告法）：
  - 经营主体发布自身名称、商标、经营范围等基本信息，未推销商品/服务
  - 自然人在自建平台发布他人商品信息（无委托证据）
  - 自然人通过用户评价、跟帖评论形式发布（无推销目的证据）
  - 招聘招募、招生简章、寻人启事、个人二手交易等非经营性信息
  - 新闻报道（不含与报道关联的商品地址/链接）
  - 知识介绍、科普宣传（不含关联商品/服务信息）

### 0.2 媒介性 — 依托大众传播媒介
- [ ] 是否通过报纸/期刊/广播/电视/电影/互联网/印刷品/广告设施/交通工具等媒介发布？
- [ ] 以下情形**不认定**为媒介性：
  - 经营场所内面对面推销
  - 现场咨询导购、上门推销
  - 展览会/博览会/推介会等现场活动
  - 线下讲座/培训

### 0.3 受众不特定性 — 可触达广泛不确定个体
- [ ] 广告受众在触达前是否为可确定的个体？
- [ ] 以下情形**认定**为受众特定（不适用广告法）：
  - 电话/短信/即时通讯点对点交流
  - 互联网聊天群内推销
  - 仅注册会员可见的信息
- [ ] 使用算法推荐/精准投放 → 仍认定受众不特定，**适用**广告法

### 0.4 非强制性 — 非法律强制展示事项
- [ ] 是否为依法须展示/标示/告知的事项？
- [ ] 以下情形**认定**为强制性（不适用广告法）：
  - 依法须在商品包装/说明书/标签标明的信息
  - 公共企事业单位依法公开信息
  - 店招牌匾等经营主体基本信息
  - 为保障消费者知情权的实物/图片展示

**结论**：
- 如不满足四性中任意一项 → **不构成商业广告**，本审查工具不适用。输出「非广告」结果（附简要理由）并终止审查。如需其他法律合规帮助（如知识产权、合同审查等），请说明具体需求。
- 如满足全部四性 → 继续 Step 1。

### 0.5 边界模糊引导 — 信息不足以判断时怎么办

> 无法独立判断四性时，向用户提问（以下场景的提示问题见 [references/GUIDANCE.md](references/GUIDANCE.md#step-0-边界模糊引导)）。

**操作规则**：无法判断 → 向用户提问 → 仍无法判断 → 输出「边界不确定」UNSURE → 用户选"从严"则默认构成广告进 Step 1。

## Step 2 — 图片与视觉内容专项审查

> 如果待审查内容中包含图片（或视频画面），通过本节检查视觉元素中的合规风险。
> 纯文本内容可跳过此步。图片审查有两种方式，取决于 AI 模型是否支持多模态视觉能力：
> - **支持多模态的模型**（如 GPT-4o、Claude 3.5 Sonnet/Vision、Gemini 2.0 Pro 等）：可直接上传图片文件，模型会分析画面内容
> - **纯文本模型**：需用户提供画面的文字描述（如"图片是一张白大褂医生微笑站立的照片，背景是医院走廊"），描述越详细审查越准确

### 2.1 画面禁用元素（对应 §9）
- [ ] 画面中是否出现国旗、国徽、军旗、军徽、党徽等国家象征？（§9(1)）
- [ ] 画面中是否出现国家机关、国家机关工作人员形象（含漫画/剪影）？（§9(2)）
- [ ] 画面中是否出现淫秽、色情、赌博、迷信、恐怖、暴力内容？（§9(8)）
- [ ] 画面中是否包含民族、种族、宗教、性别歧视性表达？（§9(9)）
- [ ] 画面是否损害国家尊严、社会公共利益或妨碍社会安定？（§9(4)/§9(5)/§9(7)）

### 2.2 画面文字审查（对应 §9(3)/§28）
- [ ] 画面中嵌入的文字是否含绝对化用语（"国家级""最高级""最佳""第一""唯一"等）？
- [ ] 画面中嵌入的数据/统计信息是否有虚构或误导嫌疑？
- [ ] 画面中的图表是否经过篡改/比例失真，具有误导性？

### 2.3 画面暗示与误导（对应 §28 虚假广告）
- [ ] 使用产品使用前后对比图，是否真实、可验证？
- [ ] 通过画面构图/特效/角度夸大产品效果或尺寸？
- [ ] 使用患者形象、医疗场景、手术画面暗示治疗效果？（医疗广告尤为敏感）
- [ ] 画面中的"效果展示"是否存在虚构或无法实现的承诺？

### 2.4 比较广告与诋毁（对应 §12 反不正当竞争法）
- [ ] 画面中直接出现竞争对手品牌/产品/商标？
- [ ] 通过画面暗示（如打叉、灰化、对比）贬低他人商品/服务？

### 2.5 包装/装潢混淆（对应 §7 反不正当竞争法）
- [ ] 画面中的商品包装、装潢是否与他人知名商品构成近似，引人误认？
- [ ] 画面中是否使用与他人有一定影响的商标、名称、域名近似标识？

### 2.6 医疗广告画面特别限制（对应 医疗广告管理办法 §7）
- [ ] 画面中是否出现疾病名称、诊疗方法、医疗技术？（§7(1)）
- [ ] 是否利用患者、医务人员、医学科研机构形象作证明？（§7(6)）
- [ ] 是否使用解放军、武警部队名义或形象？（§7(7)）
- [ ] 画面是否对医疗机构环境、设备进行主观性推介展示？（认定指南 §6(1)）

> **输入方式**：提交图片时，请附上画面描述（如"图片是一张白大褂医生微笑站立的照片，背景是医院走廊，画面右上角有'根治糖尿病'文字"）。描述越详细，审查越准确。

## Step 3 — 通用禁用情形（广告法 §9-§10）

详见 [广告法 §9-§10](references/广告法.md#第二章广告内容准则)。

### §9 绝对禁止（罚款 20万-100万起）

- [ ] **§9(1)** 使用或变相使用国旗、国歌、国徽、军旗、军歌、军徽
- [ ] **§9(2)** 使用或变相使用国家机关、国家机关工作人员名义或形象
- [ ] **§9(3)** 使用"国家级"、"最高级"、"最佳"等绝对化用语
- [ ] **§9(4)** 损害国家尊严/利益，泄露国家秘密
- [ ] **§9(5)** 妨碍社会安定，损害社会公共利益
- [ ] **§9(6)** 危害人身/财产安全，泄露个人隐私
- [ ] **§9(7)** 妨碍社会公共秩序或违背社会良好风尚
- [ ] **§9(8)** 含淫秽、色情、赌博、迷信、恐怖、暴力内容
- [ ] **§9(9)** 含民族、种族、宗教、性别歧视内容
- [ ] **§9(10)** 妨碍环境、自然资源或文化遗产保护
- [ ] **§9(11)** 法律、行政法规禁止的其他情形

**§9(3) 绝对化用语提示：** "国家级"、"最高级"、"最佳"为明令禁止。"第一"、"顶级"、"极品"、"唯一"、"100%"等需有可验证依据。已注册商标中的用语、行业标准中的等级用语可豁免。

### §10 特殊人群保护

- [ ] 是否损害未成年人身心健康
- [ ] 是否损害残疾人身心健康

## Step 4 — 引证与专利审查（广告法 §11-§12）

详见 [广告法 §11-§12](references/广告法.md#第二章广告内容准则)。

### §11 引证内容
- [ ] 数据、统计资料、调查结果等是否真实、准确
- [ ] **是否标明出处**（关键数据无出处 = BLOCK）
- [ ] 有适用范围和有效期限的 — 是否明确表示
- [ ] 涉及行政许可的 — 是否与许可内容一致

### §12 专利标注
- [ ] 是否**标明专利号和专利种类**
- [ ] 是否**谎称取得专利权**（禁止）
- [ ] 是否使用已终止、撤销、无效的专利

## Step 5 — 虚假广告 + 虚假宣传（广告法 §28 + 反不正当竞争法 §9）

详见 [广告法 §28](references/广告法.md#第二章广告内容准则) 和 [反不正当竞争法 §9](references/反不正当竞争法.md#第二章不正当竞争行为)。

### 5.1 虚假广告认定（广告法 §28）
- [ ] **§28(1)** 商品或服务不存在
- [ ] **§28(2)** 性能/功能/产地/质量/成分/价格/有效期限等信息与实际情况不符，对购买行为有实质性影响
- [ ] **§28(3)** 使用虚构/伪造/无法验证的科研成果、统计资料、调查结果等作证明材料
- [ ] **§28(4)** 虚构使用商品或接受服务的效果
- [ ] **§28(5)** 其他欺骗、误导消费者的情形

### 5.2 虚假宣传审查（反不正当竞争法 §9）
- [ ] **§9(1)** 对其商品的性能/功能/质量/销售状况/用户评价/曾获荣誉作虚假或引人误解的商业宣传？
- [ ] **§9(2)** 通过组织虚假交易、虚假评价等方式帮助他人进行虚假宣传？

> **交叉适用规则**：反不正当竞争法 §9 中的虚假宣传行为，属于发布虚假广告的 → 依照广告法 §55 处罚。不构成广告的虚假宣传 → 依照反不正当竞争法 §25 处罚。

### 罚款标准
- 虚假广告（广告法 §55）：广告费 3-5 倍；无法计算罚 20万-100万。两年内三次或严重：5-10 倍或 100万-200万，可吊销执照
- 虚假宣传（反不正当竞争法 §25）：罚 100万以下；严重 100万-200万，可吊销执照

## Step 6 — 反不正当竞争法专项审查

详见 [反不正当竞争法](references/反不正当竞争法.md)。

### 6.1 混淆行为（§7）
- [ ] 擅自使用他人有一定影响的商品名称/包装/装潢等相同或近似标识？
- [ ] 擅自使用他人有一定影响的企业名称/姓名（含笔名、艺名、网名）？
- [ ] 擅自使用他人有一定影响的域名/网站/新媒体账号/APP名称/图标？
- [ ] 将他人商标/驰名商标设置为搜索关键词，引人误认？
- [ ] 帮助他人实施混淆行为？

### 6.2 有奖销售（§11）
- [ ] 所设奖的种类/兑奖条件/奖金金额/奖品信息不明确，影响兑奖？
- [ ] 有奖销售活动开始后无正当理由变更奖的种类/条件/金额/奖品？
- [ ] 谎称有奖或故意让内定人员中奖？
- [ ] **抽奖式有奖销售最高奖金额超过5万元？**（红线！）

### 6.3 商业诋毁（§12）
- [ ] 编造、传播虚假信息或误导性信息，损害其他经营者的商业信誉/商品声誉？

### 6.4 网络不正当竞争（§13）
- [ ] 未经同意在其网络产品中插入链接/强制跳转？
- [ ] 误导/欺骗/强迫用户修改/关闭/卸载其他经营者产品？
- [ ] 恶意对其他经营者产品实施不兼容？
- [ ] 以欺诈/胁迫/破坏技术管理措施方式获取/使用其他经营者数据？
- [ ] 利用平台规则指使他人对其他经营者实施虚假交易/虚假评价/恶意退货？

## Step 7 — 特殊类别专项审查（广告法 §15-§27）

详见 [广告法 §15-§27](references/广告法.md#第二章广告内容准则)。

### 7.1 医疗、药品、医疗器械广告（§16）
- [ ] 不得含功效/安全性的断言或保证
- [ ] 不得说明治愈率或有效率
- [ ] 不得与其他药品/医疗器械/医疗机构比较
- [ ] 不得利用广告代言人作推荐、证明
- [ ] 药品广告内容是否与批准说明书一致
- [ ] 处方药广告是否标注"本广告仅供医学药学专业人士阅读"
- [ ] 非处方药广告是否标注"请按药品说明书或者在药师指导下购买和使用"

### 7.2 保健食品广告（§18）
- [ ] 不得含功效/安全性的断言或保证；不得涉及疾病预防、治疗功能
- [ ] 不得声称或暗示为保障健康所必需；不得与药品、其他保健食品比较
- [ ] 不得利用广告代言人作推荐、证明
- [ ] 是否显著标明"本品不能代替药物"

### 7.3 其他特殊类别快速检查

| 类别（条款） | 关键禁令 |
|------|---------|
| 处方药（§15） | 仅限指定专业刊物；特殊药品（麻/精/毒/放）不得做广告 |
| 烟草（§22） | 禁止在大众媒介/公共场所/公交/户外发布 |
| 酒类（§23） | 不得诱导饮酒、出现饮酒动作、表现驾驶、暗示消除紧张等 |
| 教育培训（§24） | 不得对升学/通过考试作保证性承诺；不得暗示考试机构参与 |
| 招商投资（§25） | 须有风险提示；不得保本/保收益承诺 |
| 房地产（§26） | 不得承诺升值/投资回报；不得以到达时间表示位置 |
| 种子/种畜禽（§27） | 不得作科学上无法验证的断言；不得作表示功效的断言 |

## Step 8 — 互联网广告专项审查（《互联网广告管理办法》）

详见 [互联网广告管理办法](references/互联网广告管理办法.md)。

### 8.1 互联网禁止/限制（§6-§7）
- [ ] **§6(1)**: 利用互联网发布禁售/禁发产品/服务广告？
- [ ] **§6(2)**: 发布烟草（含电子烟）广告？（**全面禁止**）
- [ ] **§6(3)**: 发布处方药广告？（**全面禁止**，法律另有规定除外）
- [ ] **§7(1)**: 特殊产品（医疗/药品/医疗器械/农药/兽药/保健食品/特医食品）— 发布前是否经审查？
- [ ] **§7(2)**: 是否严格按照审查通过的内容发布，不得剪辑、拼接、修改？

### 8.2 变相发布禁止（§8）
- [ ] 以介绍健康、养生知识等形式变相发布医疗/药品/医疗器械/保健食品/特医食品广告？
- [ ] 健康/养生知识页面是否同时出现相关地址、联系方式、购物链接？（禁止同页面出现）

### 8.3 广告可识别性（§9）
- [ ] 广告是否具有可识别性？
- [ ] **竞价排名** — 是否显著标明"广告"？
- [ ] **知识介绍/体验分享/消费测评**附购物链接的 — 是否显著标明"广告"？

### 8.4 弹出/开屏广告（§10）
- [ ] **没有关闭标志**或计时结束才能关闭？
- [ ] 关闭标志虚假/不可清晰辨识/难以定位？
- [ ] **须两次以上点击**才能关闭？
- [ ] 关闭后**继续弹出**？
- [ ] 开屏广告同样适用以上规定

### 8.5 欺骗误导点击禁止（§11）
- [ ] 虚假的系统更新/报错/清理/通知提示？
- [ ] 虚假的播放/暂停/停止/返回标志？
- [ ] 虚假的奖励承诺？

### 8.6 未成年人保护（§12）
- [ ] 在针对未成年人的互联网媒介上，是否发布医疗/药品/保健食品/特医食品/医疗器械/化妆品/酒类/美容广告或不良网络游戏广告？

### 8.7 广告主责任（§13）
- [ ] 广告主是否对内容真实性负责？
- [ ] 自行发布广告的 — 是否建立广告档案（保存不少于三年）？

### 8.8 经营者/发布者义务（§14-§15）
- [ ] 是否查验登记广告主身份信息，建立档案（保存不少于三年）？
- [ ] 是否查验证明文件、核对广告内容？
- [ ] 是否配备熟悉广告法规的审核人员？
- [ ] 用**算法推荐**发布广告的 — 是否将规则和投放记录记入档案？

### 8.9 平台经营者义务（§16）
- [ ] 是否记录保存发布广告的用户身份信息（保存不少于三年）？
- [ ] 是否监测排查违法广告，采取通知改正/删除/屏蔽等措施？
- [ ] 是否建立投诉举报受理机制？
- [ ] 是否以技术手段阻挠、妨碍市场监管部门监测？（禁止）
- [ ] 是否配合调查、保存证据？

### 8.10 搜索与发送规范（§17）
- [ ] 在搜索**政务服务**网站/APP等结果中插入竞价排名广告？（禁止）
- [ ] 未经用户同意向其交通工具、导航设备、智能家电发送广告？（禁止）
- [ ] 在用户电子邮件或即时通讯中附加广告或链接？

### 8.11 链接广告审核（§18）
- [ ] 含链接的互联网广告 — 是否核对**下一级链接**中相关广告内容？

### 8.12 直播带货（§19）
- [ ] 商品销售者通过直播推销构成商业广告的 — 是否承担广告主责任？
- [ ] 直播间运营者提供广告设计/制作/代理/发布服务的 — 是否承担广告经营者/发布者责任？
- [ ] 直播营销人员以自己名义推荐/证明构成广告代言的 — 是否承担广告代言人责任？

## Step 9 — 医疗广告认定分析（《医疗广告认定指南》2025）

详见 [医疗广告认定指南](references/医疗广告认定指南.md)。

### 9.1 定义与主体（§2）
医疗广告 = 利用一定媒介 **直接或间接** 介绍医疗机构或医疗服务的广告。
**仅依法设立的医疗机构**可发布医疗广告。

### 9.2 不构成医疗广告的情形（§3-§5, §7）
- **§3**: 医疗科研招募（受试者、临床研究患者）
- **§4**: 分诊/互联网诊疗咨询
- **§5**: 医疗机构信息公开（概况、学科、医务人员、项目、流程、医保、价格），不触发 §6
- **§5-2**: 电商平台表格化信息（必要的就医信息），不触发 §6
- **§7**: 医疗科普宣传，不触发 §8

### 9.3 信息公示 → 认定为医疗广告（§6）
- [ ] **§6(1)**: 对就医环境、医疗器材进行**主观性推介**
- [ ] **§6(2)**: 对诊疗技术/流程/效果进行**主观评价或保证性承诺**
- [ ] **§6(3)**: **与其他医疗机构比较**
- [ ] **§6(4)**: 其他以推介为目的的信息

### 9.4 科普宣传 → 变相医疗广告（§8）
- [ ] **§8(1)**: 宣称诊疗技术优势/硬件设备优势/诊疗效果
- [ ] **§8(2)**: 暗示在具体医疗机构就医有更好的安全性/疗效/价格
- [ ] **§8(3)**: 直接或间接推介具体医疗服务
- [ ] **§8(4)**: 以病例/案例方式推介
- [ ] **§8(5)**: 在科普页面附加跳转入口/购买链接
- [ ] **§8(6)**: 其他推介情形

### 9.5 错误处理（§9）
内容出现错误但不构成广告的 → 不属于广告法调整，可通报卫健部门责令整改。

### 9.6 "情节严重"认定（§12）
- 致人死亡/重伤/毁容等严重危及生命健康
- 涉嫌构成虚假广告罪
- 为无证非医疗机构引流
- 一年内 2 次以上违法医疗广告造成严重不良社会影响

## Step 10 — 医疗广告管理办法专项（2007施行）

详见 [医疗广告管理办法](references/医疗广告管理办法.md)。

### 10.1 发布前审查
- [ ] §3: 发布前是否取得《医疗广告审查证明》
- [ ] §5: 发布主体是否为依法设立的医疗机构（非医疗机构不得发布）
- [ ] §5: 是否以内部科室名义发布（禁止）

### 10.2 允许内容清单（§6）— 仅限 8 项
- (1) 医疗机构第一名称 (2) 地址 (3) 所有制形式 (4) 医疗机构类别
- (5) 诊疗科目 (6) 床位数 (7) 接诊时间 (8) 联系电话
- (1)-(6) 须与《医疗机构执业许可证》一致

### 10.3 禁止表现形式（§7）
- [ ] **§7(1)**: 涉及医疗技术、诊疗方法、疾病名称、药物
- [ ] **§7(2)**: 保证或隐含保证治愈
- [ ] **§7(3)**: 宣传治愈率、有效率
- [ ] **§7(4)**: 淫秽、迷信、荒诞
- [ ] **§7(5)**: 贬低他人
- [ ] **§7(6)**: 利用患者/医务人员/医学科研机构作证明
- [ ] **§7(7)**: 使用解放军和武警部队名义

### 10.4 发布规范
- [ ] §14: 是否标注医疗机构第一名称和《医疗广告审查证明》文号
- [ ] §16: 是否利用新闻形式/医疗资讯节目变相发布（禁止）
- [ ] §16: 人物专访中出现医疗机构名称时，是否同时出现地址/联系方式？（禁止）
- [ ] §16: 是否在同一时段/版面发布广告

输出必须遵循 [references/REPORT_TEMPLATE.md](references/REPORT_TEMPLATE.md) 模板（含基本信息、BLOCK/WARN/INFO/UNSURE 四档分类表、违规等级评估、UNSURE 处理规则）。**审查完毕后再读 Gotchas 做最终检查。**

## 审查验证

完成审查后，在输出报告前执行以下验证：

1. **自检映射**：确认每一项发现（BLOCK/WARN/INFO/UNSURE）都对应到具体法规条款和原文位置，不得笼统描述
2. **完整性检查**：确认 13 步流程中每一步都已执行完毕，没有跳步
3. **边界复查**：如果你在审查过程中对任何结论感到不确定 → 标记为 UNSURE 并在报告中说明缺少什么信息
4. **修正建议可操作性**：每条修改建议应明确指出改什么、改成什么，不能写"请合规"之类的空话
5. **所有 UNSURE 项数量超过 3 项** → 建议用户补充信息后重新提交
